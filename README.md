# YT-PlaylistDownloader



### How to install

- make sure you have nodejs installed

- Download [here]https://github.com/FlorianElke/YT-PlaylistDownloader/archive/1.0.zip or just simply git clone https://github.com/FlorianElke/YT-PlaylistDownloader.git

- cd YT-PlaylistDownloader 

- npm install

- npm start

- enjoy
